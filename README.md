# YambaMama

## Project Description
YambaMama is a mobile application designed to help users track their menstrual cycles and understand their health better. The app provides various features like cycle predictions, reminders, and health tips to support users in managing their menstrual health.

## Features
- **Cycle Tracking:** Keep track of menstrual cycles and ovulation days.
- **Reminder Notifications:** Set reminders for upcoming periods and fertility windows.
- **Health Insights:** Access informative articles and tips related to menstrual health.
- **Data Privacy:** Securely store and manage personal health data. 

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Mariah-Miranda/YambaMama.git
   ```
2. Navigate into the project directory:
   ```bash
   cd YambaMama
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Tech Stack
- **Frontend:** React Native
- **Backend:** Node.js
- **Database:** MongoDB
- **Version Control:** Git
